Detailed Description

A helper class used in ApplicationWindow to set your app's theme colors.

Properties

accentColor : string

The accent theme color used in your app for a highlight color in many UI components. This is a helper property to set Theme::accentColor.

backgroundColor : string

The default background color used throughout the app. This is a helper property to set Theme::backgroundColor.

primaryColor : string

The primary theme color used in your app for the toolbar and other primary UI elements. This is a helper property to set Theme::primaryColor.

primaryDarkColor : string

A darker version of the primary theme color used by the system status bar and window decorations. This is a helper property to set Theme::primaryDarkColor.
